# IDE Setup Instructions

## PyCharm Setup
1. Open PyCharm.
2. Go to `File` -> `New Project`.
3. Choose a project type (e.g., Pure Python).
4. Specify the location of your project.
5. Click on `Create`.
6. Once the project is created, go to `File` -> `Settings` (or `Preferences` on macOS).
7. In the left sidebar, click on `Project: <Project Name>` and select `Python Interpreter`.
8. Click on the gear icon and choose `Add...`.
9. Select `Virtualenv Environment` and create a new environment or choose an existing one.
10. Install the necessary packages using the terminal inside PyCharm or through the PyCharm interface.

## VSCode Setup
1. Open Visual Studio Code.
2. Install the Python extension from the Extensions Marketplace.
3. Open the command palette (Ctrl + Shift + P or Cmd + Shift + P) and type `Python: Select Interpreter`.
4. Choose your Python interpreter from the list.
5. Create a new folder for your project or open an existing one.
6. Create a new Python file in the folder.
7. Install any necessary packages using the terminal in VSCode with `pip install <package>`.

## Conclusion
Now you are all set up to start coding!
